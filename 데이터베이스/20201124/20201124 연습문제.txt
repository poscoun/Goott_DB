1.   사원들의 이름, 부서 번호, 부서 이름을 출력하시오
SQL> SELECT E.ENAME, E.DEPTNO, D.DNAME
  2  FROM EMP E, DEPT D;


2.   30번 부서의 사원들의 이름, 직업, 부서위치를 출력하시오
SQL> SELECT E.ENAME, E.JOB, D.LOC
  2  FROM EMP E LEFT OUTER JOIN DEPT D
  3  ON (E.DEPTNO = D.DEPTNO)
  4  WHERE E.DEPTNO = 30;

SQL> SELECT E.ENAME, E.JOB, D.LOC
  2  FROM EMP E, DEPT D
  3  WHERE E.DEPTNO = D.DEPTNO
  4  AND E.DEPTNO = 30;

3.  수당을 받는 사원의 이름 , 부서이름 및 부서위치를  하시오
(단 수당이 0인사원도 포함)

SQL> SELECT E.ENAME, D.DNAME, D.LOC
  2  FROM EMP E, DEPT D
  3  WHERE E.DEPTNO = D.DEPTNO
  4  AND COMM >= 0;

SQL> SELECT E.ENAME, D.DNAME, D.LOC
  2  FROM EMP E JOIN DEPT D
  3  ON E.DEPTNO = D.DEPTNO
  4  AND COMM >= 0;

4.   DALLAS 에서 근무하는 사원의 이름 , 직업, 부서번호 , 부서이름
     을 출력하시오
SQL> SELECT E.ENAME, E.JOB, E.DEPTNO, D.DNAME
  2  FROM EMP E, DEPT D
  3  WHERE E.DEPTNO = D.DEPTNO
  4  AND D.LOC = 'BOSTON';

SQL> SELECT E.ENAME, E.JOB, E.DEPTNO, D.DNAME
  2  FROM EMP E JOIN DEPT D
  3  ON E.DEPTNO = D.DEPTNO
  4  AND D.LOC = 'BOSTON';


5.   이름에  A  가  들어가는 사원들의 이름과 부서이름을 출력하시오

SQL> SELECT E.ENAME, D.DNAME
  2  FROM EMP E, DEPT D
  3  WHERE E.DEPTNO = D.DEPTNO
  4  AND E.ENAME LIKE '%A%';

SQL> SELECT E.ENAME, D.DNAME
  2  FROM EMP E JOIN DEPT D
  3  ON E.DEPTNO = D.DEPTNO
  4  AND E.ENAME LIKE '%A%';

6.  사원이름과    그 사원의 관리자 이름을 출력하시오
( 단 컬럼HEADING을   employeee,  manager  출력 )
SQL> SELECT E.ENAME "EMPLOYEEE", C.ENAME "MANGER"
  2  FROM EMP E, EMP C
  3  WHERE E.MGR = C.EMPNO(+);

SQL> SELECT E.ENAME "EMPLOYEEE", C.ENAME "MANGER"
  2  FROM EMP E JOIN EMP C
  3  ON E.MGR = C.EMPNO(+);


7.    사원이름과 직업, 급여, 급여등급 을 출력하시오

SQL> SELECT E.ENAME, E.SAL, S.GRADE
  2  FROM EMP E, SALGRADE S
  3  WHERE E.SAL BETWEEN S.LOSAL AND S.HISAL;

SQL> SELECT E.ENAME, E.SAL, S.GRADE
  2  FROM EMP E INNER JOIN SALGRADE S
  3  ON E.SAL BETWEEN S.LOSAL AND S.HISAL;


8.   사원이름과 부서명과 월급을 출력하는데 월급이 3000 이상인 사원을
     출력하시오
SQL> SELECT E.ENAME, D.DNAME, E.SAL
  2  FROM EMP E, DEPT D
  3  WHERE E.DEPTNO = D.DEPTNO
  4  AND E.SAL >= 3000;

SQL> SELECT E.ENAME, D.DNAME, E.SAL
  2  FROM EMP E JOIN DEPT D
  3  ON E.DEPTNO = D.DEPTNO
  4  AND E.SAL >= 3000;


9. 사원의 이름, 부서번호 와 같은 부서에 근무하는 동료 사원들을
출력하시오
 ex)       FORD  20 SMITH
           JONES  20 SMITH
           JAMES  30 ALLEN
                 …
SQL> SELECT E.ENAME, C.DEPTNO, C.ENAME
  2  FROM EMP E, EMP C
  3  WHERE E.DEPTNO = C.DEPTNO
  4  AND E.ENAME != C.ENAME(+);

SQL> SELECT E.ENAME, C.DEPTNO, C.ENAME
  2  FROM EMP E JOIN EMP C
  3  ON E.DEPTNO = C.DEPTNO
  4  AND E.ENAME != C.ENAME(+);

10. BLAKE 이란 사원보다 늦게 입사한 사원의 이름과 입사일을 출력하시오

SELECT E.ENAME, E.HIREDATE
FROM EMP E, EMP C
WHERE C.ENAME = 'BLAKE' AND E.HIREDATE > C.HIREDATE;

SELECT E.ENAME, E.HIREDATE
FROEM EMP E JOIN EMP C
ON C.ENAME = 'BLAKE' AND E.HIREDATE > C.HIREDATE;


SQL> SELECT ENAME, HIREDATE
  2  FROM EMP
  3  WHERE HIREDATE > (SELECT HIREDATE
  4     FROM EMP
  5     WHERE ENAME = 'BLAKE');
